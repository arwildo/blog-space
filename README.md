## Requirements
* Python3
* Django

## Installing

1. Clone the repo

```
git clone https://github.com/arwildo/blog-space
```

2. add your SECRET_KEY to blogSpace/.local_settings

```python
def key():
    return "SECRET_KEY"
```

3. Start the server

```
python3 manage.py runserver
```
